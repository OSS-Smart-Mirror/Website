Project Overview:

The intended design of the project incorporates a smart computer interface that provides cosmetic accessories to the user in the form of a digitally enhanced mirror. The design can be fit onto any flat surface or hung from a wall, and in addition to simply serving as a mirror, will present a set of users the option to view metrics important to their daily needs like indoor temperature, weather, a news feed, and a user interface that will allow users to try out accessories like hats, mustaches, and popular image filters, all controllable through the usage of an infrared gesture system.

Project Specific Success Criteria:

  1. PSSC #1: An ability to successfully read ambient conditions using a temperature and light sensor using ADC, GPIO
  2. PSSC #2: An ability to perform user gesture detection using an Infrared Sensor on the STM using ADC
  3. PSSC #3: An ability to successfully communicate gestures readings, ambient light and temperature values between the STM and the Jetson through USART
  4. PSSC #4: An ability to authenticate the user using a facial recognition API using the Jetson
  5. PSSC #5: An ability to display metrics and relevant information (email, weather, stock tickers, news) using a dashboard on the Jetson Monitor
